Flowfy é uma plataforma de gestão de projetos técnicos desenvolvida como parte de um projeto acadêmico, com foco em organização, produtividade e colaboração entre equipes de tecnologia.

Sobre o Projeto

O Flowfy foi idealizado para auxiliar desenvolvedores, líderes técnicos e gestores de produto a organizarem tarefas, acompanharem o progresso de projetos e visualizarem informações de forma clara e eficiente.

A proposta do sistema é simular uma ferramenta moderna de gestão de tarefas, inspirada em soluções do mercado, com foco em experiência do usuário (UX) e organização visual.

Objetivos

Criar, editar e organizar tarefas técnicas
Atribuir responsáveis e prazos
Visualizar status e progresso dos projetos
Identificar gargalos e prioridades
Melhorar a clareza na gestão de múltiplas demandas
Público-Alvo

Desenvolvedores
Líderes técnicos
Gestores de produto
Equipes de tecnologia em geral
Funcionalidades

Principais

Criação de tarefas

Atribuição de responsáveis

Definição de prazos

Alteração de status (ex: Em andamento, Concluído) Secundárias

Edição e reatribuição de tarefas

Filtros por status, prioridade e responsável

Visualização detalhada de tarefas

Histórico de alterações e comentários

Interface e UX

O projeto foi desenvolvido com foco em:

Organização de grandes volumes de informação
Leitura rápida e tomada de decisão
Feedback visual claro para ações do usuário
Interface limpa e moderna
Design

O design segue um estilo minimalista e funcional, utilizando:

Layout estruturado para desktop
Uso de cores para indicar estados e ações
Componentes reutilizáveis
Menu lateral interativo (abrir/fechar)
Ferramentas Utilizadas

Figma (prototipação e design de interface)
Interações

O protótipo inclui interações como:

Menu lateral colapsável
Navegação entre telas
Simulação de ações do usuário
Contexto Acadêmico

Este projeto foi desenvolvido como parte de um exercício acadêmico na área de Tecnologia da Informação, com foco em:

UX/UI Design
Prototipação
Estruturação de sistemas
